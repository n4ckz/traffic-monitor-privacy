Politique de Confidentialité

Dernière mise à jour : Novembre 2025

1. Introduction

Traffic Monitor ("nous", "notre", "l'application") respecte votre vie privée. Cette politique de confidentialité explique quelles informations nous collectons et comment nous les utilisons.

2. Données collectées

L'application collecte les données suivantes :

  • Coordonnées GPS des tronçons : Les points de départ et d'arrivée que vous saisissez pour surveiller le trafic.
  • Préférences d'alertes : Les plages horaires et jours que vous configurez.
  • Noms des tronçons : Les noms que vous attribuez à vos tronçons.

3. Utilisation des données

Vos données sont utilisées exclusivement pour :

  • Interroger les services de trafic (TomTom) pour vous fournir l'état de la circulation
  • Sauvegarder vos préférences pour une utilisation ultérieure

4. Partage des données

Nous ne vendons pas vos données personnelles. Les coordonnées GPS sont transmises à TomTom uniquement pour obtenir les informations de trafic. Consultez la politique de confidentialité de TomTom.

5. Stockage des données

Vos données sont stockées sur nos serveurs sécurisés. Vous pouvez supprimer vos tronçons à tout moment depuis l'application.

6. Données de localisation

L'application n'accède PAS à votre position GPS en temps réel. Vous saisissez manuellement les coordonnées des tronçons à surveiller.

7. Vos droits

Conformément au RGPD, vous avez le droit de :

  • Accéder à vos données
  • Rectifier vos données
  • Supprimer vos données
  • Exporter vos données

8. Contact

Pour toute question concernant cette politique de confidentialité, contactez-nous à : contact@trafficmonitor.app

9. Modifications

Nous pouvons mettre à jour cette politique de confidentialité. Les modifications seront publiées sur cette page.
